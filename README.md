# Next.js Live News App

A simple news application built using Next.js App Router. It fetches the latest English news from the NewsData API and displays them in clean, card-based UI with images, titles, descriptions, and published dates.

## Features

* Live English news fetched from NewsData API
* Server-side data fetching using Next.js
* Responsive card layout
* About page
* Styled using custom CSS
* Fast performance with SSR

* Next.js 16 (App Router)

## Setup

```
git clone https://github.com/hardhikbangera/newsAPI.git
npm install
npm run dev
```

Open in browser:

```
http://localhost:3000
```
